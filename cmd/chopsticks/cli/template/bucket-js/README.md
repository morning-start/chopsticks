# {{.Name}}

A Chopsticks Bucket (JavaScript).

## Install Dependencies

```bash
npm install
```

## Add to Chopsticks

```bash
chopsticks source add {{.Name}} /path/to/{{.Name}}
```

## Development

```bash
# Create a new app
chopsticks bucket create my-app

# Test installation
chopsticks install my-app --bucket {{.Name}}
```
