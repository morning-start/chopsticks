## 任务管理流程

### 任务文档存放

临时任务文档存放在 `temp/` 目录下，包含以下子目录：

- `temp/tasks/` - 任务文档存放
- `temp/reports/` - 报告文档存放

该目录已被添加到 `.gitignore`，不会进入版本控制。

### 分支管理规范

处理任务时遵循以下分支流程：

1. **创建任务分支**

   ```bash
   git checkout -b task-<任务名称>
   ```

2. **处理任务**
   - 在任务分支上进行开发
   - 任务文档可存放在 `temp/tasks/` 目录下
   - 报告文档可存放在 `temp/reports/` 目录下

3. **完成任务后**
   ```bash
   git checkout main
   git merge task-<任务名称>
   git branch -d task-<任务名称>
   ```

### 注意事项

- 每个任务都应创建独立的分支
- 任务文档仅用于临时记录，不提交到 git
- 合并到主干前确保代码通过测试
